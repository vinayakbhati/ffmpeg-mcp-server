# Integrate mcp as tool to VS Code
1. Open VS Code
2. Navigate to AI Chat box (like github Copilot)
3. Press Cntrl+Shift+P
4. Select MCP:Add Server 
5. Select Http Option
6. place local path url at which mcp server is up
7. Add name of mcp server
8. Now, you can connected to MCP Server :)
9. Verify it with Configure Plugin
10. Now, annoate the server and start Prompting
